# Twins - High-Fidelity Persona Extraction

**nC2で恋リアをやりたいと思ったことはない？**

デジタルツインを使った恋愛シミュレーションアプリケーション。論文「Love First, Know Later」に基づく実装。

---

## 🎯 コンセプト

「型（Type）ではなく、個（Individual）を通じて、相性を理解し未来をシミュレートする。」

既存の静的情報（年収、身長）や性格診断（MBTI、BIG5）を排除し、ユーザーの「デジタルツイン（分身）」を生成。相手の分身との「恋リア」を通じて、実際に付き合った後にしか分からない相性を事前に予測します。
